## Инжектор чата Nebulous.io

Программа сначала перехватывает пакет сообщения из игры (для этого можно отправить любое сообщение в чат)
После этого можно общаться через консоль, не заходя напрямую в чат игры, вводя нужные сообщения через `chat_cli.py`

***

Запуск:

```python
python chat_cli.py
```

Есть вложенная инструкция `/help`
