## Инжектор чата Nebulous.io

Программа сначала перехватывает UDP пакет сообщения из игры (для этого можно отправить любое сообщение в чат)
После этого можно отправлять свои chat-сообщения без открытия чата в игре,
используя настоящий сетевой контекст клиента через `chat_cli.py`

***

Запуск:

```python
python chat_cli.py
```

Есть вложенная инструкция `/help`

## Как работает

Frida-хук подключается к системным функциям:

`send`
`sendto`

Когда игра отправляет UDP-пакет, агент проверяет:

`packet[0] == 0x89`

Это opcode chat-пакета

После обнаружения пакет разбирается:

```
0x00    opcode (0x89)
0x01-04 session / unknown bytes
0x05-06 nickLen (u16 BE)
...     nickname
...     msgLen (u16 BE)
...     message
...     tail (служебные данные)
```

Строки хранятся как:

```
u16 (big-endian) + UTF-8 bytes
```
Пакет отправляется через тот же сокет:

```
sendto(fd, ...)   // если есть sockaddr
или
send(fd, ...)     // fallback
```

Tail пакета не трогается
`... ff ff ff ff 00 00 ...`

Он содержит служебные данные: flags, id, состояние клиента и т.д.
